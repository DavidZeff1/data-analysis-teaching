Dashboards

What Are Dashboards?

Dashboards combine multiple worksheets into a single interactive view. They tell a complete data story by showing related visualizations together. Well-designed dashboards:

  • Answer business questions - "How is sales performance this quarter?"
  • Enable exploration - Click one chart to filter others
  • Provide context - See the big picture and details together
  • Drive action - Clear insights lead to decisions

Key Intuitions

  • Start with your story - What question does this dashboard answer?
  • Build worksheets first - Create all your charts before the dashboard
  • The funnel icon is your friend - Fastest way to add filter actions
  • Use containers for structure - Makes resizing and reorganizing easier
  • Test, test, test - Click everything to make sure filters work as expected
  • Less is more - A focused dashboard beats a cluttered one

Practice Exercises

  2. Create a new dashboard and add all 3 charts
  3. Add a title using the Text object
  4. Make the Region chart act as a filter for the other charts
  5. Add an interactive Category filter that viewers can use
  6. Add a Year filter to let viewers select time periods
  7. Experiment with Horizontal and Vertical containers to change the layout
  8. Try different dashboard sizes (fixed, automatic, range)
  9. Add a company logo using the Image object
  10. Create a second dashboard and add navigation buttons between them
