Building a Dashboard

1. Set size
   Dashboard ‚Üí Size ‚Üí Fixed or Automatic
   (1200x800 is common)

2. Drag sheets to canvas
   Drag from Sheets panel to canvas
   Blue highlight shows where it will land

3. Arrange layout
   Use containers (Horizontal/Vertical) for structure
   Resize by dragging edges

4. Add interactivity
   Click sheet ‚Üí Use as Filter (funnel icon)
   This makes clicking one chart filter others

5. Add elements
   Drag Text for titles
   Drag Image for logos

Best Practices

  • Start with a clear story or question to answer
  • Most important viz goes top-left (first thing seen)
  • 3-4 charts per dashboard max - avoid clutter
  • Use consistent colors across charts
  • Include context: titles, labels, filters visible
  • Test interactivity - click around to verify filters work

Practice

  1. Create 3 worksheets: Sales by Region (bar), Sales over Time (line), Profit by Category (bar)
  2. Create a new dashboard and add all 3 charts
  3. Add a title using the Text object
  4. Make the Region chart act as a filter for the others
  5. Add an interactive Category filter that applies to all sheets
