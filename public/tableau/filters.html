Filters

What they do

Filters limit the data shown in your visualization. You can filter by dimensions, measures, or dates.

Types of Filters

Dimension Filter  - Keep/exclude specific values
                    "Show only East and West regions"

Measure Filter    - Filter by numeric range
                    "Show only Sales > 1000"

Date Filter       - Filter by time period
                    "Show only 2017 data"

Context Filter    - Filter that applies first
                    "Top 10 within East region only"

Adding Filters

// Method 1: Drag to Filters shelf
Drag Region to Filters shelf
→ Select which regions to include

// Method 2: Right-click in view
Right-click a bar → Exclude (or Keep Only)

// Method 3: From Data pane
Right-click field → Show Filter
→ Creates interactive filter card

Filter Card Styles

Right-click filter card to change style:

Single Value (list)     - Radio buttons
Single Value (dropdown) - Dropdown menu
Multiple Values (list)  - Checkboxes
Multiple Values (dropdown) - Multi-select dropdown
Wildcard               - Text search
Slider                 - For measures/dates

Filter Examples

// Show only specific categories
Drag Category to Filters
→ Check: Furniture, Technology

// Show sales above threshold
Drag Sales to Filters
→ Range: 500 to maximum

// Show last 2 years
Drag Order Date to Filters
→ Relative dates → Last 2 years

// Top 10 products by sales
Drag Product Name to Filters
→ Top tab → By field: Top 10 by Sales

Filter Order of Operations

1. Extract Filters
2. Data Source Filters
3. Context Filters      ← Use for "Top N within..."
4. Dimension Filters
5. Measure Filters
6. Table Calc Filters

Key intuitions

  • Filters on the shelf don't show UI - use "Show Filter" for interactive
  • Right-click filter → Apply to Worksheets → All Using This Data Source
  • Context filters are needed for correct "Top N" within a subset
  • Use "Exclude" sparingly - "Include" is usually clearer
  • Quick filters (filter cards) let users interact with your viz
