Chart Types

Choosing the Right Chart

The chart type you choose should match your analytical question. Each visualization is best suited for specific types of comparisons and patterns. Tableau's "Show Me" panel helps, but understanding when to use each chart makes you more effective.

Bar Chart - Comparing Categories

Best for: Comparing values across discrete categories

How to Build:

// Horizontal bar chart
Rows: Category
Columns: SUM(Sales)

// Vertical bar chart (column chart)
Rows: SUM(Sales)
Columns: Category

// Sorted by value
Click sort icon in toolbar
OR right-click axis → Sort
Tech
Furn
Office

When to use: Comparing sales by region, profit by product, count by category. Use horizontal bars when labels are long (state names, product names).

Line Chart - Trends Over Time

Best for: Showing how values change over a continuous period

How to Build:

// Basic line chart
Columns: Order Date (continuous)
Rows: SUM(Sales)

// Change date granularity
Right-click date pill → select:
  Year, Quarter, Month, Week, Day

// Multiple lines (by category)
Color: Category

// Tip: Green date pill = continuous
// Blue date pill = discrete
Technology Furniture

When to use: Sales trends, growth over time, seasonality patterns. Time should always go on the X-axis (left to right = past to future).

Scatter Plot - Relationships Between Variables

Best for: Exploring correlation between two numeric measures

How to Build:

// Sales vs Profit by Product
Columns: SUM(Sales)
Rows: SUM(Profit)
Detail: Sub-Category

// Add trend line
Analytics pane → drag "Trend Line"

// Size by quantity
Size: SUM(Quantity)

// Color by category
Color: Category
Each dot = one sub-category

When to use: Finding correlations (discount vs profit), identifying outliers, comparing two metrics simultaneously. Great for "Does X affect Y?" questions.

Pie & Donut Charts - Parts of a Whole

Best for: Showing composition when you have 2-5 categories

How to Build:

// Using Show Me
1. Select Region and Sales
2. Click Show Me → Pie chart

// Manual method
Marks type: Pie
Angle: SUM(Sales)
Color: Region
Label: Region, SUM(Sales)

// Show percentages
Right-click pill → Quick Table Calc
  → Percent of Total

⚠️ Warning: Pie charts are hard to read with more than 5 slices. For many categories, use a bar chart instead. Humans are bad at comparing angles!

Dual Axis - Two Measures, One View

Best for: Comparing two related metrics with different scales

How to Build:

// Sales (bars) and Profit (line)
Columns: Order Date
Rows: SUM(Sales)

// Add second measure
Drag SUM(Profit) to right Y-axis
  (or right-click → Dual Axis)

// Synchronize axes (important!)
Right-click right axis → Synchronize Axis

// Different mark types
Click SUM(Sales) in Marks → Bar
Click SUM(Profit) in Marks → Line

When to use: Sales vs Profit over time, Revenue vs Orders, any two metrics you want to compare on the same time axis.

Best for: Showing patterns across two dimensions with color intensity

Choosing Charts - Quick Reference

Question Chart Type
How do categories compare?Bar chart
How does it change over time?Line chart
What's the composition?Pie chart (≤5 categories) or Stacked bar
Is there a correlation?Scatter plot
How do two dimensions interact?Heatmap / Highlight table
Where is it located?Map
Compare two metrics over time?Dual axis

Key Intuitions

  • Show Me is helpful - it highlights which charts work with your selected fields
  • Dates on Columns usually = line chart - time flows left to right
  • Dimension on Rows + Measure on Columns = bar chart
  • Two measures + Detail = scatter plot
  • Right-click everything - there are always more options
  • Sort descending - almost always sort bar charts by value
  • Dual axis needs synchronizing - or the comparison is misleading

Practice Exercises

  1. Create a bar chart: Profit by Sub-Category, sorted descending
  2. Create a line chart: Sales by Month, with separate lines for each Region
  3. Create a scatter plot: Discount (X) vs Profit (Y), colored by Category
  4. Create a dual axis chart: Sales (bars) and Quantity (line) over time
  5. Create a heatmap: Sales by State and Category
  6. Create a pie chart: Sales by Region with percentages
  7. Try the Show Me panel - select Category and Sales, click through different chart types
  8. Create a stacked bar chart: Sales by Region, stacked by Segment
  9. Add a trend line to your scatter plot (Analytics pane)
  10. Create a bar-in-bar chart: Sales and Profit by Category (dual axis, both as bars)
