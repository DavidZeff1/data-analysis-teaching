Chart Types

Common Charts & How to Build

Bar Chart

// Sales by Category
Rows: Category
Columns: SUM(Sales)

// Sorted descending
Click sort icon in toolbar (or right-click axis)

Line Chart

// Sales over time
Columns: Order Date (as continuous)
Rows: SUM(Sales)

// Change date granularity
Right-click date pill → Month, Quarter, Year

// Multiple lines by category
Color: Category

Scatter Plot

// Sales vs Profit by Sub-Category
Columns: SUM(Sales)
Rows: SUM(Profit)
Detail: Sub-Category

// Add trend line
Analytics pane → Drag "Trend Line" to view

Pie Chart

// Sales by Region
1. Click Show Me → Pie chart
2. Or manually:
   Marks type: Pie
   Angle: SUM(Sales)
   Color: Region
   Label: Region, SUM(Sales)

Dual Axis

// Sales (bars) and Profit (line)
Columns: Order Date
Rows: SUM(Sales)
      SUM(Profit)  (drag second measure to Rows)

Right-click second axis → Dual Axis
Right-click axis → Synchronize Axis

Change mark types separately:
Click SUM(Sales) dropdown → Bar
Click SUM(Profit) dropdown → Line

Highlight Table (Heatmap)

// Sales by Region and Category
Rows: Region
Columns: Category
Color: SUM(Sales)
Label: SUM(Sales)

Marks type: Square (for cleaner look)

Key intuitions

  • Show Me highlights valid chart types for your data
  • Dates on Columns usually = line chart
  • Dimension on Rows + Measure on Columns = bar chart
  • Two measures on axes + dimension on Detail = scatter plot
  • Right-click for more options (sort, format, filter)

Practice

  1. Create a bar chart: Profit by Sub-Category, sorted descending
  2. Create a line chart: Sales by Month, with separate lines for each Region
  3. Create a scatter plot: Discount vs Profit, colored by Category
  4. Create a dual axis chart: Sales and Quantity over time
  5. Create a heatmap: Sales by State and Category
