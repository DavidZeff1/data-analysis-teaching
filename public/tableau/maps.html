Maps

What they do

Tableau automatically recognizes geographic data and can create maps. Great for showing regional patterns and location-based insights.

Geographic Fields

Tableau auto-detects:
- Country
- State/Province
- City
- Postal Code / ZIP
- Latitude / Longitude

Look for the globe icon üåê next to fields

Creating a Map

// Method 1: Auto-map
Double-click a geographic field (State, City, etc.)
‚Üí Tableau creates a map automatically!

// Method 2: Manual
Columns: Longitude (generated)
Rows: Latitude (generated)
Detail: State (or other geo field)

// Method 3: Show Me
Select geo field + measure
Click Show Me ‚Üí Symbol Map or Filled Map

Map Types

Symbol Map     - Circles on locations
                Size by Sales, Color by Profit

Filled Map     - Regions filled with color
                States colored by Sales total

Density Map    - Heatmap showing concentration
                Where are orders clustered?

Change type: Marks dropdown ‚Üí Map, Circle, etc.

Map Examples

// Sales by State (filled)
Double-click State
Marks type: Map (filled)
Color: SUM(Sales)

// City performance (symbol)
Double-click City
Marks type: Circle
Size: SUM(Sales)
Color: SUM(Profit)

// Profit/Loss by state
Double-click State
Color: SUM(Profit)
‚Üí Edit Colors ‚Üí Diverging palette (red-green)

Customizing Maps

// Change background
Map ‚Üí Map Layers
‚Üí Style: Light, Dark, Streets, Satellite

// Show/hide elements
Map ‚Üí Map Layers
‚Üí Toggle: Coastline, State borders, Cities

// Zoom to area
Map ‚Üí Map Options ‚Üí Pan and Zoom (or Fixed)
Use selection tools to zoom

// Fix unknown locations
Click "X unknown" indicator
Edit Locations ‚Üí Match to correct place

Key intuitions

  • Tableau needs to recognize geo fields - check for globe icon
  • Right-click field ‚Üí Geographic Role to set/change type
  • Filled maps work best for regions (states, countries)
  • Symbol maps work best for points (cities, stores)
  • Use diverging colors for profit/loss (red/green or orange/blue)
  • Maps are great for "where" questions in dashboards

Practice
