Maps

Why Maps?

Geographic visualizations reveal spatial patterns that tables and charts miss. Maps answer "where" questions:

  • Where are sales concentrated?
  • Which regions underperform?
  • How does performance vary by state/city?
  • Are there geographic clusters or patterns?

Tableau automatically recognizes geographic data and knows where places are on a map.

Creating Maps

// Method 1: Auto-map (fastest!)
Double-click a geographic field (State, City, etc.)
‚Üí Tableau creates a map automatically!

// Method 2: Drag to canvas
Drag State to the canvas
‚Üí Tableau plots points on a map

// Method 3: Using Show Me
Select State and Sales
Click Show Me ‚Üí Symbol Map or Filled Map

// Method 4: Manual
Columns: Longitude (generated)
Rows: Latitude (generated)
Detail: State

Map Types

Symbol Map (Proportional)

Circles on locations, sized/colored by values

Best for: Points (cities, stores, customers)

Filled Map (Choropleth)

Regions filled with color based on values

Best for: Regions (states, countries, territories)

Building Map Examples

// Filled map: Sales by State
Double-click State
Marks type: Map (filled regions)
Color: SUM(Sales)
‚Üí States colored by total sales

// Symbol map: Cities sized by sales
Double-click City
Marks type: Circle
Size: SUM(Sales)
Color: SUM(Profit)
‚Üí Bubbles: bigger = more sales, color = profit

// Profit/Loss by State
Double-click State
Color: SUM(Profit)
‚Üí Edit Colors ‚Üí Diverging palette (red-green)
‚Üí Red = loss, Green = profit

Click the indicator showing "X unknown"

Color Palettes for Maps

Choose colors that communicate effectively:

Sequential - Low to high (sales, quantity)
Diverging - Negative to positive (profit/loss)
Categorical - Different groups (regions, segments)

To change: Click Color on Marks card ‚Üí Edit Colors ‚Üí Choose palette

  • Misspelled city/state names
  • Non-standard abbreviations
  • Missing country context for ambiguous city names

Fix: Click the indicator ‚Üí Edit Locations ‚Üí Manually match unrecognized places

Key Intuitions

  • Double-click is fastest - double-click any geo field to auto-create a map
  • Globe icon = geographic - Tableau already knows it's a location
  • Filled maps for regions, symbol maps for points
  • Diverging colors for +/- - use red/green or orange/blue for profit/loss
  • Add State to Detail for city maps - helps resolve ambiguous city names
  • Maps work great in dashboards - use them as filters for other charts
  • Check for unknowns - always verify all locations are mapped correctly
