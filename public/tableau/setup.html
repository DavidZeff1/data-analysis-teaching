Practice Setup

What is Tableau?

Tableau is the industry-leading data visualization tool. It transforms raw data into interactive dashboards through a drag-and-drop interface - no coding required. Tableau excels at:

  • Speed - Create complex visualizations in minutes, not hours
  • Interactivity - Dashboards respond to clicks, filters, and selections
  • Visual exploration - Discover patterns by experimenting with different views
  • Sharing - Publish dashboards for others to explore

1. Get Tableau Public (Free)

Go to public.tableau.com and download Tableau Public.

Installation Steps:

  1. Click Create (top right corner)
  2. Sign up for a free Tableau Public account
  3. Download the installer for your operating system
  4. Run the installer and follow the prompts
  5. Launch Tableau Public and sign in

üí° Tableau Public vs Tableau Desktop

Tableau Public is fully featured - the only limitation is that all your work is saved publicly to the Tableau Public gallery. For learning, this is perfect! Tableau Desktop (paid) allows private saves and connects to more data sources.

2. Load the Sample Superstore Dataset

Great news - Tableau includes the Superstore dataset by default! This is the same data we used in Excel.

Loading the Data:

Alternative: Under "Saved Data Sources" click "Sample - Superstore" if available.

3. Alternative: Use Web Authoring

If you don't want to install software, you can use Tableau directly in your browser:

  1. Go to public.tableau.com
  2. Sign in and click Create ‚Üí Web Authoring
  3. Upload the Superstore CSV from this link

Web authoring has fewer features than the desktop app, but works for learning the basics.

4. Understanding the Dataset

The Superstore dataset simulates a fictional retail company selling office supplies, furniture, and technology across the United States. Each row represents one product in an order.

üìä Dimensions (Blue Pills)

Categorical data - used for grouping and filtering

  • Order ID - Unique order number
  • Order Date - When ordered
  • Ship Mode - Shipping speed
  • Customer Name - Buyer name
  • Segment - Consumer/Corporate/Home Office
  • Region - East/West/Central/South
  • Category - Product category
  • Sub-Category - Detailed type

üìà Measures (Green Pills)

Numeric data - used for calculations

  • Sales - Revenue amount
  • Quantity - Units sold
  • Discount - Discount applied
  • Profit - Profit (can be negative)

Measures are automatically aggregated (SUM, AVG, etc.) when you use them.

6. Verify Your Setup

When you reach the worksheet, you should see:

  • Left panel shows Dimensions (blue) and Measures (green)
  • Fields include: Order Date, Ship Mode, Segment, Category, Sales, Profit, etc.
  • Empty canvas in the center waiting for your visualization
  • Marks card on the left for customizing appearance

Try double-clicking "Sales" in the Measures area - Tableau will create a simple bar chart automatically!
