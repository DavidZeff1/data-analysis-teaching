💡 What Tableau is for

Tableau is built around one loop: drag a field, see a picture, ask the next question. That speed of iteration is the point — you explore by trying things rather than by specifying them.

⚡ Speed

A chart in three drags. Exploring twenty questions takes minutes.

🖱️ Interactivity

Dashboards respond to clicks. Viewers answer their own follow-up questions.

👁️ Visual exploration

Patterns you'd never find in a table become obvious in a chart.

📤 Sharing

Publish a live dashboard rather than emailing a screenshot.

1️⃣ Get Tableau Public (free)

  1. Go to the download page

    Visit public.tableau.com and choose Create in the top-right corner.

  2. Create a free account

    You'll need one to save work. This also becomes your public portfolio — a real advantage when job hunting.

  3. Download and install

    Pick the installer for macOS or Windows and run it. It's a large download; start it before you make coffee.

  4. Launch and sign in

    You'll land on the Connect screen, which is where the next section starts.

Tableau Public vs Tableau Desktop

Public is fully featured for everything in this course. Two limitations matter: your workbooks save publicly to the Tableau Public gallery, and it can't connect to databases — only files and a few web sources. For learning, neither gets in the way. Never put confidential data in it.

2️⃣ Load the Superstore dataset

Tableau ships with Superstore built in — the same fictional retailer used throughout this site.

Easiest: the bundled copy
  1. Open Tableau Public
  2. Under Connect → To a File, click More…
  3. Go to Documents → My Tableau Repository → Datasources
  4. Open Sample - Superstore.xls

Sometimes it's listed directly under "Saved Data Sources" on the start screen.

Alternative: the CSV
  1. Download the CSV (right-click → Save Link As)
  2. Connect → To a File → Text file
  3. Select the downloaded file

Use this if the bundled version is missing. It has one table rather than three.

No install possible? Tableau Public also runs in the browser: sign in at public.tableau.com → Create → Web Authoring, then upload the CSV. Fewer features, but the basics all work.

4️⃣ Verify your setup

Click Sheet 1 at the bottom of the window. You should see:

  • A field list on the left, split into blue dimensions and green measures
  • Columns and Rows shelves across the top
  • A Marks card on the left with Color, Size, Label, Detail and Tooltip
  • An empty canvas in the middle
  • A Show Me panel collapsed at the top right
Your first chart, right now

Double-click Sales, then double-click Category. Tableau builds a bar chart on its own. That two-double-click loop is the fastest way to explore a new dataset, and it's worth doing on every field before you plan anything.

📖 Terms on this page

Defined before they're used. Every term on the site is also in the glossary.

Worksheet
One sheet holding one chart. The tabs along the bottom of the window.
Example: Sheet 1 is the worksheet you land on after connecting.
Field
One column of your data, listed down the left in the Data pane.
Example: Superstore has 21 fields: Order Date, Region, Sales, Profit…
Dimension
A field you group by — categories, names, dates. Shown in blue.
Example: Region, Category, Customer Name. Four regions means four rows.
Measure
A number you can add up. Shown in green, and aggregated automatically.
Example: Sales, Profit, Quantity, Discount.
Aggregate
To roll many rows up into one number by summing or averaging.
Example: 9,994 order lines aggregate into one $2,297,201 total.
Data pane
The list of all your fields down the left-hand side of the window.
Example: after connecting, it lists Order Date, Region, Sales and the rest — blue ones on top, green below.
Marks card
The panel just below the Data pane that controls how your data points look — their colour, size and labels.
Example: dropping Region onto its Color button colours your bars by region.
Show Me
A panel that suggests chart types based on the fields you've selected.
Example: select Region and Sales and it highlights bar chart, map, and a few others as valid choices.
📍 Button at the top right of the toolbar
Granularity
How finely the data is chopped up — what a single row actually represents.
Example: one Superstore row is one product inside one order. An order containing three products takes up three rows.
