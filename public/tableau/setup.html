Practice Setup

1. Get Tableau Public (Free)

Go to public.tableau.com

  1. Click Create (top right)
  2. Sign up for a free account
  3. Download and install Tableau Public

Tableau Public is fully featured - the only limitation is all work is saved publicly.

2. Load the Sample Superstore dataset

Good news - Tableau includes it by default!

  1. Open Tableau Public
  2. Under Connect, click More...
  3. Select Sample - Superstore

This is the same dataset we used in Excel - a fictional retail company's sales data.

3. Alternative: Use Web Authoring

4. Dataset Overview

The Superstore dataset contains:

Dimensions (Categories):

  • Order ID, Order Date, Ship Date
  • Customer Name, Segment
  • City, State, Region, Country
  • Product Name, Category, Sub-Category
  • Ship Mode

Measures (Numbers):

  • Sales
  • Quantity
  • Discount
  • Profit

5. Verify it works

After connecting to the data:

  • You should see "Orders" table with ~10,000 rows
  • Left panel shows dimensions (blue) and measures (green)
  • Drag "Sales" to the canvas - you should see a bar
