Filters & Slicers

Filter Types

Visual-level   - Affects only one visual
Page-level     - Affects all visuals on the page
Report-level   - Affects all pages in the report
Drillthrough   - Navigate to detail page

Using the Filters Pane

// Add a filter
1. Expand Filters pane (right side)
2. Drag field from Fields pane to filter area
3. Choose filter level:
   - "Filters on this visual"
   - "Filters on this page"
   - "Filters on all pages"

// Filter options
Basic: Check/uncheck values
Advanced: Contains, starts with, is blank, etc.
Top N: Top 10 by Sales

Slicers (Interactive Filters)

// Create a slicer
1. Click Slicer icon in Visualizations
2. Drag a field to the Field well
   Example: Region

// Slicer styles (Format pane)
- List (vertical checkboxes)
- Dropdown
- Between (for dates/numbers)
- Relative date (last 7 days, this month)
- Tile (horizontal buttons)

Slicer Examples

// Category dropdown
Slicer field: Category
Format → Slicer settings → Options → Dropdown

// Date range slider
Slicer field: Order Date
Format → Slicer settings → Options → Between

// Region buttons
Slicer field: Region
Format → Slicer settings → Options → Tile
General → Orientation → Horizontal

Cross-Filtering

Visuals interact automatically!

When you click a bar in one chart:
- Other visuals filter/highlight
- This is called "cross-filtering"

Control interaction type:
Format → Edit interactions
Choose per visual: Filter, Highlight, or None

Sync Slicers Across Pages

// Make slicer affect multiple pages
1. Select the slicer
2. View → Sync slicers
3. Check which pages to sync
4. Check which pages to show the slicer

Key intuitions

  • Filters pane = hidden filters (for report builders)
  • Slicers = visible filters (for report users)
  • Click visual, then add to "Filters on this visual" for visual-specific
  • Ctrl+Click in slicers to multi-select
  • Slicers with dates auto-detect and offer relative options
  • Use "Edit interactions" to control which visuals respond

Practice

  1. Create a Region slicer as horizontal buttons
  2. Create a Category slicer as a dropdown
  3. Create a date slicer showing a date range slider
  4. Add a page-level filter for Segment = "Consumer"
  5. Use Edit interactions to make one chart NOT filter another
