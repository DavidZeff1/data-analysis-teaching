Visualizations

Common Visual Types

Bar & Column Charts

// Clustered Bar - horizontal
Y-axis: Category
X-axis: Sales

// Clustered Column - vertical
X-axis: Category
Y-axis: Sales

// Stacked - add a Legend field
Legend: Region
→ Bars split by region

Line Chart

// Sales over time
X-axis: Order Date
Y-axis: Sales

// Multiple lines
Legend: Category

// Change date granularity
Click arrow on date field in well
→ Choose: Year, Quarter, Month, Day

Pie & Donut Charts

// Sales by Region
Legend: Region
Values: Sales

// Show percentages
Format → Detail labels → Label contents
→ Select "Percent of total"

Card & KPI

// Card - single big number
Fields: Sales
→ Shows total sales

// Multi-row Card - multiple metrics
Fields: Sales, Profit, Quantity

// KPI - with target/trend
Value: Sales
Trend axis: Order Date
Target: Target Sales (if you have it)

Table & Matrix

// Table - flat list
Columns: Product Name, Sales, Profit

// Matrix - pivot table style
Rows: Category, Sub-Category
Columns: Region
Values: Sales
→ Expandable hierarchy!

Map

// Bubble map
Location: State (or City)
Size: Sales
Color: Profit

// Filled map
Location: State
Color saturation: Sales

Note: May need to set data category
Select column → Column tools → Data category → State

Scatter Chart

// Sales vs Profit by Sub-Category
X-axis: Sales
Y-axis: Profit
Details: Sub-Category

// Add size dimension
Size: Quantity

// Add trend line
Analytics pane → Trend line → Add

Key intuitions

  • Hover over visual icons to see names
  • Click a visual, then click different icon to change type
  • Some visuals need specific field types (dates for line charts)
  • Format pane (paint roller) customizes appearance
  • Analytics pane adds trend lines, reference lines, etc.

Practice

  1. Create a column chart: Profit by Sub-Category
  2. Create a line chart: Sales by Month, with lines for each Region
  3. Create a Card showing total Sales
  4. Create a Matrix: Rows = Category/Sub-Category, Columns = Region, Values = Sales
  5. Create a Map showing Sales by State
