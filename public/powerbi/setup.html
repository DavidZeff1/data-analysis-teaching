Practice Setup

1. Get Power BI Desktop (Free)

Download from powerbi.microsoft.com/desktop

Power BI Desktop is completely free. Only publishing to Power BI Service requires a license.

Note: Windows only. Mac users can use a VM or Power BI Service (web).

2. Alternative: Power BI Service (Web)

No download needed:

  1. Go to app.powerbi.com
  2. Sign up with a work/school email (free tier available)
  3. Create reports directly in browser

3. Load the Sample Superstore dataset

  1. Download the CSV from this link (right-click → Save As)
  2. Open Power BI Desktop
  3. Click Get Data → Text/CSV
  4. Select the downloaded file
  5. Click Load

4. Dataset Overview

Same Superstore dataset used in Excel and Tableau:

Dimensions (Text):

  • Order ID, Order Date, Ship Date
  • Customer Name, Segment
  • City, State, Region, Country
  • Product Name, Category, Sub-Category
  • Ship Mode

Measures (Numbers):

  • Sales
  • Quantity
  • Discount
  • Profit

5. Verify it works

After loading data:

  • You should see "superstore" in the Fields pane (right side)
  • Expand it to see all columns
  • Drag "Sales" to the canvas - a bar chart should appear

Power BI vs Tableau

Tableau              →  Power BI
Dimensions/Measures  →  Fields
Worksheets           →  Report Pages
Calculated Fields    →  DAX Measures
Marks Card           →  Format Pane
Show Me              →  Visualizations Pane
