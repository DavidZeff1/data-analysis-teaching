Practice Setup

1. Open the compiler

Go to sqliteonline.com

2. Load Chinook database

  1. Click Syntax (top right)
  2. Select Example: Chinook SQL
  3. Click Run

3. Available Tables

  • Artist - artist names
  • Album - albums linked to artists
  • Track - songs with price, duration, genre
  • Customer - customer info and location
  • Invoice - purchases with dates and totals
  • InvoiceLine - line items per invoice
  • Employee - staff with manager hierarchy
  • Genre, MediaType, Playlist, PlaylistTrack

4. Test it

Clear the editor and run:

SELECT * FROM Artist LIMIT 5;
