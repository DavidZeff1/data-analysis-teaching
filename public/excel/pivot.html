📊 Pivot Tables

The single most powerful tool for spreadsheet analysis. Summarize thousands of rows into meaningful totals — and reorganize them instantly by dragging fields.

⏱️ 30 min ⭐ Core skill 🔗 Needs: Tables, Cleaning

💡 What pivot tables do

  • Summarize thousands of rows into totals
  • Group data by category (region, product, date)
  • Calculate sums, averages, counts, and more
  • Reorganize instantly by dragging fields
  • Filter to focus on specific segments

What would take dozens of SUMIF formulas can be done in seconds with a pivot table.

🛠️ Creating a pivot table

Google Sheets
  1. Select any cell in your data
  2. Insert → Pivot table
  3. Choose "New sheet" (recommended)
  4. Click Create
Excel
  1. Select any cell in your data
  2. Insert → PivotTable
  3. Verify the range and location
  4. Click OK

🎛️ The four drop zones

📊 Rows

Categories as row labels (vertical). e.g. Region, Category, Customer

📈 Columns

Categories across columns (horizontal). e.g. Year, Quarter, Segment

🔢 Values

Numbers to calculate (SUM, AVG, COUNT). e.g. Sales, Profit, Quantity

🔍 Filters

Limit what's included. e.g. only 2017, only East

🔢 Changing the calculation type

By default numbers are summed. To change it:

Google Sheets: value field dropdown → Summarize by → SUM, COUNT, AVERAGE, MAX, MIN…

Excel: right-click a value → Value Field Settings → choose calculation

SUM — total of values
COUNT — number of items
AVERAGE — mean value
MAX — highest value
MIN — lowest value
COUNTUNIQUE — distinct count

📅 Grouping dates

Add a date field to Rows, then group it by year, quarter, or month:

Google Sheets: date field dropdown → "Group rule" → Year / Month / Quarter…

Excel: right-click a date → Group → choose level

This turns thousands of daily rows into clean monthly or yearly summaries.

↕️ Sorting, filtering & percentages

// Sort by values
Row-label dropdown → Sort by → Values (ascending/descending)

// Filter specific items
Row-label dropdown → uncheck items to hide

// Top 10
Add a filter field → Filter by condition → Top 10

Show as % of total: Sheets → value dropdown → "Show as" → % of grand total. Excel → right-click value → Show Values As → % of Grand Total.

➕ Multiple values at once

Add the same field several times with different calculations:

  • Drag Sales to Values (SUM)
  • Drag Sales again → change to AVERAGE
  • Drag Sales again → change to COUNT

Now you see total, average, and number of sales in one pivot.

⚠️ Three traps worth knowing

Counting rows instead of orders

Each Superstore row is a line item, so "Count of Order ID" overstates orders by about 5×. Use Distinct Count in the Value Field Settings instead.

Averaging an average

The average of four regional profit margins is not the overall margin — each region has a different sales base. Compute SUM(Profit)/SUM(Sales) instead.

Stale numbers

A pivot shows a snapshot from the last refresh. Change the source and the pivot does not update until you right-click → Refresh. Build the source as a Table so new rows are at least included when you do.

✓ Pivot to explore, formulas to report

Find what matters with a pivot, then rebuild the handful of numbers that matter with SUMIFS — those cells stay where you put them and never need refreshing. That's how the dashboard project is built.

🧠 Key intuitions

  • Pivots don't change your data — they create a summary view
  • Refresh to update — right-click → Refresh after data changes
  • Text → Rows/Columns, numbers → Values
  • Start simple — one row field, one value field, then add complexity
  • Double-click a cell to drill into the underlying rows
  • Multiple row fields create hierarchy — expandable groups
  • Filter instead of deleting — toggle segments on/off easily
