Essential Excel Shortcuts

Mastering these shortcuts will significantly speed up your workflow. The Command key (⌘) on Mac is roughly equivalent to the Control key (Ctrl) on Windows.

Action Windows Mac
General File Operations
New Workbook Ctrl + N ⌘ + N
Open Workbook Ctrl + O ⌘ + O
Save Ctrl + S ⌘ + S
Navigation
Move to edge of data region Ctrl + Arrow Keys ⌘ + Arrow Keys
Move one screen right Alt + PgDn Fn + ⌥ + ↓
Move one screen left Alt + PgUp Fn + ⌥ + ↑
Formatting
Format Cells Dialog Ctrl + 1 ⌘ + 1
Bold Ctrl + B ⌘ + B
Number Format Ctrl + Shift + ! Ctrl + Shift + !
Currency Format Ctrl + Shift + $ Ctrl + Shift + $
Percentage Format Ctrl + Shift + % Ctrl + Shift + %
Formulas & Editing
Edit Active Cell F2 F2 (or Ctrl + U)
Toggle Absolute Ref ($) F4 ⌘ + T (or F4)
Autosum Alt + = ⌘ + Shift + T
Paste Special Ctrl + Alt + V ⌘ + Ctrl + V
Selection
Select All Ctrl + A ⌘ + A
Select Entire Row Shift + Space Shift + Space
Select Entire Column Ctrl + Space Ctrl + Space
Extend Selection to Edge Ctrl + Shift + Arrow ⌘ + Shift + Arrow
Select Current Region Ctrl + Shift + * ⌘ + Shift + *
Select Visible Cells Only Alt + ; ⌘ + Shift + Z
Add to Selection (non-adjacent) Ctrl + Click ⌘ + Click
Insert & Delete
Insert Row/Column/Cell Ctrl + Shift + + ⌘ + Shift + +
Delete Row/Column/Cell Ctrl + - ⌘ + -
Clear Cell Contents Delete Delete or Fn + ⌫
Data Entry
Fill Down Ctrl + D ⌘ + D
Fill Right Ctrl + R ⌘ + R
Enter in Multiple Cells Ctrl + Enter ⌘ + Enter
Insert Current Date Ctrl + ; Ctrl + ;
Insert Current Time Ctrl + Shift + ; ⌘ + ;
Flash Fill (auto-pattern) Ctrl + E ⌘ + E
New Line in Cell Alt + Enter ⌥ + Enter
Go To & Find
Go To (cell/range) Ctrl + G or F5 Ctrl + G
Go To Beginning (A1) Ctrl + Home Ctrl + Fn + ←
Go To Last Used Cell Ctrl + End Ctrl + Fn + →
Find Ctrl + F ⌘ + F
Find & Replace Ctrl + H Ctrl + H
Worksheets
Next Sheet Ctrl + PgDn ⌥ + →
Previous Sheet Ctrl + PgUp ⌥ + ←
Insert New Sheet Shift + F11 Shift + Fn + F11
Rows & Columns
Hide Rows Ctrl + 9 ⌘ + 9
Unhide Rows Ctrl + Shift + 9 ⌘ + Shift + 9
Hide Columns Ctrl + 0 ⌘ + 0
Unhide Columns Ctrl + Shift + 0 ⌘ + Shift + 0
Group Rows/Columns Alt + Shift + → ⌘ + Shift + K
Ungroup Rows/Columns Alt + Shift + ← ⌘ + Shift + J
Auto-fit Column Width Alt + H + O + I Double-click column border
View & Misc
Show/Hide Formulas Ctrl + ` Ctrl + `
Freeze Panes Dialog Alt + W + F + F View Menu → Freeze Panes
Add Filter Ctrl + Shift + L ⌘ + Shift + F
Create Table Ctrl + T ⌘ + T
Insert Chart Alt + F1 Fn + ⌥ + F1
Undo Ctrl + Z ⌘ + Z
Redo Ctrl + Y ⌘ + Y
Print Ctrl + P ⌘ + P
Name Box (jump to cell) Ctrl + G or click Name Box ⌘ + G or click Name Box
