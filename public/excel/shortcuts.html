Essential Excel Shortcuts

Mastering these shortcuts will significantly speed up your workflow. The Command key (⌘) on Mac is roughly equivalent to the Control key (Ctrl) on Windows.

Action Windows Mac
General File Operations
New Workbook Ctrl + N ⌘ + N
Open Workbook Ctrl + O ⌘ + O
Save Ctrl + S ⌘ + S
Navigation
Move to edge of data region Ctrl + Arrow Keys ⌘ + Arrow Keys
Move one screen right Alt + PgDn Fn + ⌥ + ↓
Move one screen left Alt + PgUp Fn + ⌥ + ↑
Formatting
Format Cells Dialog Ctrl + 1 ⌘ + 1
Bold Ctrl + B ⌘ + B
Number Format Ctrl + Shift + ! Ctrl + Shift + !
Currency Format Ctrl + Shift + $ Ctrl + Shift + $
Percentage Format Ctrl + Shift + % Ctrl + Shift + %
Formulas & Editing
Edit Active Cell F2 F2 (or Ctrl + U)
Toggle Absolute Ref ($) F4 ⌘ + T (or F4)
Autosum Alt + = ⌘ + Shift + T
Paste Special Ctrl + Alt + V ⌘ + Ctrl + V
