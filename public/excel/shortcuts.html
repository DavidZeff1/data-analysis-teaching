⌨️ Keyboard Shortcuts

The difference between someone who uses Excel and someone who is fast at Excel is almost entirely muscle memory. Learn the top ten first — they cover most of the gap — then use the searchable reference below as you need it.

⏱️ Ongoing 📖 Reference ⌘ Mac & Windows

🧠 How to actually learn these

1. Three at a time

Pick three from the top ten and force yourself to use only those this week. Trying to learn thirty at once means learning none.

2. Ban the mouse

When you catch yourself reaching for a menu you know a shortcut for, undo and do it again with the keyboard. It's slower today and much faster by Friday.

3. Learn the patterns

Shortcuts aren't random. Ctrl plus an arrow jumps; adding Shift selects instead of moving. Learn the rule, get a dozen shortcuts free.

Mac mapping: ⌘ usually stands in for Ctrl, and ⌥ for Alt. Function keys often need Fn unless you've turned that off in System Settings. Where the two platforms genuinely differ, both are listed below.

The one that changes how you work: Ctrl Shift ↓

Click the first data cell in a column, press it, and the entire column of data is selected — however many rows there are. Combine it with Ctrl Shift → to grab the whole dataset. This is how experienced users select ranges, and it never gets the boundary wrong.

💎 Three that most people never learn

F9 on part of a formula

While editing, select just one argument and press F9. Excel replaces it with its computed value so you can see what that piece actually returns. Press Esc to restore it. This is the single best debugging tool for a nested formula.

Ctrl Enter to fill a selection

Select a range, type a formula once, then press Ctrl Enter instead of Enter. It fills every selected cell at once, with references adjusting normally. Faster and more reliable than dragging.

Alt ; for visible cells only

Filter your data, select the range, press Alt ;, then copy. Only the visible rows come across — without it, copying a filtered range quietly brings the hidden rows too.

🎯 Practice drill

Twenty minutes on the Superstore data, mouse untouched. If you reach for it, start the step again.

  1. From A1, jump to the last row of data and back to the top
  2. Select the entire Sales column of data in one keystroke, and read the sum off the status bar
  3. Select the whole dataset with Ctrl Shift *
  4. Convert it to a Table, then toggle the Total row on and off
  5. Turn filters on, open the Region dropdown from the keyboard, and filter to West
  6. Copy the visible rows only, using Alt ; before copying
  7. Paste them into a new sheet as values only via Paste Special
  8. Write =R2/S2 in a new column and fill it down to the last row with Ctrl D
  9. Edit that formula and use F4 to lock a reference
  10. Format the column as currency, then as a percentage, then back to General
  11. Show all formulas with Ctrl `, then hide them again
  12. Insert a new sheet, name it, and return to the first sheet — all from the keyboard
🏆 Challenge: the no-mouse hour

Do a full piece of real work — import, clean, summarise, chart — without touching the mouse for one hour. You'll be slower and slightly frustrated. Do it three times and you'll be permanently faster than you were, because the shortcuts you looked up under pressure are the ones that stick.
