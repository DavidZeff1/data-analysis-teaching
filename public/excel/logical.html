🔀 Logical Functions

Logical functions make decisions: they evaluate a condition (true or false?) and return different results based on the answer — like building "if-then" logic into your cells.

⏱️ 25 min ⭐ Core skill 🔗 Needs: Formulas, References
On this page The idea Operators IF Nested IF IFS AND / OR IFERROR Practice

💡 The idea

Think of a flow chart: "IF the profit is positive, THEN label it 'Profit', OTHERWISE label it 'Loss'." Every logical function builds on testing a condition that resolves to TRUE or FALSE.

⚖️ Comparison operators

These compare two values and return TRUE or FALSE.

<>

5<>3 → TRUE
>

5>3 → TRUE

🏛️ IF — the foundation

IF is the most important logical function. It has three parts:

=IF( condition, value_if_true, value_if_false )
condition
A test returning TRUE or FALSE
value if true
Returned when the test is TRUE
value if false
Returned when the test is FALSE
// Is the profit positive or negative?
=IF(U2>0, "Profit", "Loss")

// Return the actual profit value or "No Profit"
=IF(U2>0, U2, "No Profit")

// 10% bonus on sales over 500, otherwise 0
=IF(R2>500, R2*0.1, 0)

// Was a discount applied?
=IF(T2>0, "Discounted", "Full Price")

🪆 Nested IF — multiple outcomes

Put an IF inside another to handle more than two outcomes:

// Categorize sales into three tiers
=IF(R2>500, "Large",
    IF(R2>100, "Medium",
        "Small"))

// Reads: if >500 → "Large"; else if >100 → "Medium"; else "Small"

Nesting more than 3 levels gets confusing fast — reach for IFS instead.

📋 IFS — cleaner multiple conditions

IFS evaluates conditions in order and returns the first TRUE result:

=IFS(
    R2>1000, "Extra Large",
    R2>500,  "Large",
    R2>100,  "Medium",
    R2>0,    "Small",
    TRUE,    "Zero or Negative"
)
// TRUE as the last condition acts as "else" — catches everything left

IFS stops at the first TRUE, evaluating top-to-bottom. Order matters — put the most specific conditions first.

🔗 AND / OR — combining conditions

AND — all must be TRUE
// Profitable AND no discount
=IF(AND(U2>0, T2=0),
    "Great sale",
    "Regular sale")
OR — any can be TRUE
// East OR West region
=IF(OR(M2="East", M2="West"),
    "Coastal",
    "Inland")
// Complex: (sales>1000 OR profit>200) AND Consumer segment
=IF(
    AND(
        OR(R2>1000, U2>200),
        H2="Consumer"
    ),
    "VIP Customer",
    "Regular Customer"
)

🔄 NOT — reverse a condition

=IF(NOT(U2>0), "Needs Review", "OK")
// Same as:
=IF(U2<=0, "Needs Review", "OK")

🛡️ IFERROR — handle errors gracefully

Some formulas can error out (e.g. dividing by zero). IFERROR catches them:

// Without IFERROR — may show #DIV/0! when S2 is 0
=R2/S2

// With IFERROR — show 0 instead
=IFERROR(R2/S2, 0)

// Show "N/A" for errors
=IFERROR(R2/S2, "N/A")

// Useful for a VLOOKUP that may not find a match
=IFERROR(VLOOKUP(...), "Not Found")

🔎 ISBLANK, ISNUMBER, ISTEXT

=IF(ISBLANK(A2), "Empty", "Has Data")
=IF(ISNUMBER(A2), "Number", "Not a Number")
=IF(ISTEXT(A2), "Text", "Not Text")

🧠 Key intuitions

  • Master IF first before moving to IFS
  • Conditions return TRUE/FALSE — try typing =5>3
  • Text needs quotes — "Profit", not Profit
  • Numbers don't — 100, not "100"
  • AND/OR go inside IF — they return TRUE/FALSE, not values
  • IFERROR is defensive programming — always use it for division and lookups
  • Order matters in IFS — most specific conditions first

Add a new column to your data for each exercise.

  1. Profit Status: "Profit" if Profit > 0, "Break Even" if = 0, "Loss" if < 0
  2. Discount Flag: "Discounted" if Discount > 0, else "Full Price"
  3. Sales Tier: "Small" (<$100), "Medium" ($100–$500), "Large" ($500–$1000), "Enterprise" (>$1000)
  4. Priority: "High" if Ship Mode is "Same Day" OR Sales > $1000, else "Normal"
  5. Coastal Region: "Yes" if Region is "East" or "West", else "No"
  6. Needs Review: "Yes" if Profit is negative AND Discount > 0
  7. Profit Margin % with IFERROR showing "N/A" for errors
  8. Customer Segment Value: 3 = Consumer, 2 = Corporate, 1 = Home Office
  9. High Value Order: "Yes" if Sales > $500 AND Quantity > 3
  10. Shipping Speed Score (use IFS): 4 = Same Day, 3 = First Class, 2 = Second Class, 1 = Standard Class
