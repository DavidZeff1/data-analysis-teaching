📌 Cell References

Understanding how references behave when you copy a formula is the single most important skill for working efficiently in spreadsheets. Get it wrong and you write a thousand formulas; get it right and you write one.

❓ Why references matter

A cell reference like R2 is the address Excel looks up to find a value. References are powerful because:

  • Formulas update automatically when data changes
  • You can copy one formula across many rows
  • You can build complex calculations that pull from many cells

🔑 Types of references

A1Relative

Shifts when copied. Write =R2*S2 in V2 →

V3 becomes =R3*S3, V4 becomes =R4*S4

✓ Use for row-by-row calculations

$A$1Absolute

Never changes. $ locks both column and row. =R2/$R$9995 →

V3 becomes =R3/$R$9995 (the total stays fixed)

✓ Use for fixed cells like totals or constants

$A1Mixed — column locked

=$A1*B$1. Copy right → =$A1*C$1 ($A stays).

Copy down → =$A2*B$1

✓ Use for multiplication tables / matrices

A$1Mixed — row locked

=A$1*$B1. Copy right → =B$1*$B1.

Copy down → =A$1*$B2

⌨️ The F4 shortcut

Instead of typing $ manually, click a reference in your formula and press F4 to cycle:

A1 → F4 → $A$1 → F4 → A$1 → F4 → $A1 → F4 → A1

📐 Worked example: % of total

A common task — calculating what percentage each row is of the total. The total stays fixed (absolute), each row's Sales changes (relative).

RowSales (R)Formula in column VResult
2$261.96=R2/$R$99950.011%
3$731.94=R3/$R$99950.032%
4$14.62=R4/$R$99950.001%
9995$2,297,201=SUM(R2:R9994)Total

$R$9995 (the total) stays fixed while R2, R3, R4 change for each row.

🔗 Referencing other sheets

=Sheet2!A1                        // Cell A1 on Sheet2
='Summary Sheet'!B5               // Quotes needed if the name has spaces
=SUM(Sheet2!R:R)                  // Sum a whole column from another sheet
=Sheet1!$A$1 + Sheet2!$A$1        // Add values from two sheets

🏷️ Named ranges

Give ranges meaningful names instead of memorizing addresses:

  1. Select the range (e.g. R2:R9994)
  2. Click in the Name Box (left of the formula bar)
  3. Type a name like SalesData and press Enter
  4. Now use =SUM(SalesData) instead of =SUM(R2:R9994)

Named ranges make formulas readable and automatically behave as absolute references.

🧠 Key intuitions

  • $ locks what follows it — $A locks the column, $1 locks the row
  • Default is relative — references change unless you add $
  • Absolute for totals — when dividing by a fixed cell
  • Relative for row calculations — when the same formula applies to each row
  • F4 toggles $ — fastest way
  • Test by copying — paste your formula elsewhere to verify it behaves

🎯 Practice exercises

  1. Calculate each row's Sales as a % of total Sales, using an absolute reference for the total. Copy down and verify.
  2. Calculate Profit Margin per row (Profit ÷ Sales) using only relative references.
  3. Create a named range AllSales for the Sales column and use it in a SUM.
  4. Put the total Sales in one cell, then reference it (absolute) to compute each row's % of total.
  5. Build a summary table on a new sheet that references totals from your data sheet.
  6. Build a 5×5 multiplication table using mixed references — write ONE formula you can copy to fill it.
  7. Calculate Price Per Unit (Sales ÷ Quantity) per row.
  8. Calculate Profit Per Unit (Profit ÷ Quantity) per row.
