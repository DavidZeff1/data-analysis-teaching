2️⃣ Import the Sample Superstore dataset

We'll use the famous "Sample Superstore" dataset — a fictional retail company's sales data. It's widely used in data analysis courses because it has realistic business data with multiple dimensions and measures.

Paste this in cell A1 and press Enter
=IMPORTDATA("https://raw.githubusercontent.com/leonism/sample-superstore/master/data/superstore.csv")

Wait 5–10 seconds for the data to load. You'll see ~10,000 rows appear.

3️⃣ Understand the dataset

The Superstore data simulates a retailer selling office supplies, furniture, and technology across the United States. Each row is a single item in an order.

📋 Dimension columns (categories)

  • Row ID — unique row identifier
  • Order ID — order number (one order can span multiple rows)
  • Order Date / Ship Date
  • Ship Mode — Standard, Second Class, First Class, Same Day
  • Customer ID, Customer Name
  • Segment — Consumer, Corporate, Home Office
  • Country, City, State, Postal Code, Region
  • Product ID, Category, Sub-Category, Product Name

📊 Measure columns (numbers)

  • Sales — revenue in dollars
  • Quantity — number of items sold
  • Discount — 0 = none, 0.2 = 20% off
  • Profit — profit or loss (can be negative!)

🔑 Questions this data can answer

  • Which products are most profitable?
  • Which regions generate the most sales?
  • Do discounts increase or decrease profit?
  • Who are the most valuable customers?
  • How do sales trend over time?

4️⃣ Verify your setup

  • Row 1 contains headers (Row ID, Order ID, Order Date, …)
  • You should have ~9,994 rows of data (plus the header row)
  • Column A is "Row ID" with numbers 1, 2, 3, …
  • Column R is "Sales" with dollar amounts
  • Scroll right — about 21 columns total

5️⃣ Alternative: Microsoft Excel Online

If you prefer Excel or need it for work:

  1. Go to Excel Online (free with a Microsoft account)
  2. Download the CSV: click here (right-click → Save Link As)
  3. In Excel Online, click "Upload and open" and select the file

IMPORTDATA doesn't work in Excel — you must download and upload the file manually.
