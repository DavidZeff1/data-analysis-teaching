🧮 Basic Formulas

Formulas turn a spreadsheet from a grid of numbers into a calculator that updates itself. Master the building blocks here before moving on.

💡 What are formulas?

Formulas are instructions that tell Excel to perform calculations. Every formula starts with an equals sign =, which signals "this is a calculation, not text."

They can be as simple as =5+3 or as complex as analyzing thousands of rows. The key idea: formulas can reference cells, so when your data changes, your results update automatically.

🔬 Anatomy of a formula

=SUM(R2:R9995)
= Starts every formula
SUM Function name
( ) Parentheses hold arguments
R2:R9995 Range of cells

∑ Essential aggregate functions

These take a range of cells and return a single value.

SUM — add all values in a range
=SUM(R2:R9995)         // Total of all Sales
=SUM(R:R)              // Entire column (slower, includes new data)
=SUM(R2:R100, U2:U100)  // Multiple ranges
AVERAGE — the arithmetic mean
=AVERAGE(R2:R9995)     // Average sale amount
=AVERAGE(U:U)          // Average profit per item

AVERAGE ignores empty cells and text, but includes zeros.

COUNT / COUNTA — count cells
=COUNT(R2:R9995)       // Count cells containing numbers
=COUNTA(B2:B9995)      // Count non-empty cells (includes text)
=COUNTBLANK(R:R)       // Count empty cells

COUNT only counts numbers. COUNTA counts any non-empty cell. Use COUNTA to count orders by Order ID.

MIN / MAX — smallest or largest value
=MIN(R2:R9995)   // Smallest sale amount
=MAX(R2:R9995)   // Largest sale amount
=MIN(U:U)        // Biggest loss (most negative profit)
=MAX(U:U)        // Biggest profit
ROUND — round to a number of decimals
=ROUND(3.14159, 2)        // 3.14
=ROUND(AVERAGE(R:R), 2)   // Average rounded to 2 decimals
=ROUNDUP(3.14, 1)         // Always up: 3.2
=ROUNDDOWN(3.99, 0)       // Always down: 3

➗ Basic arithmetic

Use standard math operators directly in formulas:

=R2 + U2          // Addition: Sales + Profit
=R2 - U2          // Subtraction: Sales - Profit (= Cost)
=R2 * S2          // Multiplication: Sales × Quantity
=R2 / S2          // Division: Sales ÷ Quantity (price per unit)
=R2 ^ 2           // Exponent: Sales squared
=(R2 + U2) / 2    // Parentheses control order of operations

Order of operations (PEMDAS): Parentheses → Exponents → Multiplication/Division → Addition/Subtraction.

⚠️ Common mistakes to avoid

❌ Wrong
SUM(R:R)

Missing equals sign

✓ Correct
=SUM(R:R)

Starts with =

❌ Wrong
=SUM(R2-R9995)

Minus instead of colon

✓ Correct
=SUM(R2:R9995)

Colon defines a range

🧠 Key intuitions

  • All formulas start with = — this tells Excel to calculate
  • R:R means the entire column R — includes current and future data
  • R2:R100 is a fixed range — faster, but won't include new rows
  • Functions can be nested — e.g. =ROUND(AVERAGE(R:R), 2)
  • Excel ignores spaces — =SUM(R:R) and = SUM( R:R ) are the same
  • #DIV/0! — you divided by zero or an empty cell
  • #VALUE! — you tried to do math on text

🎯 Practice exercises

Create a new sheet called "Summary" and calculate the following — compare your answers to verify.

  1. Total Sales for all orders (≈ $2.3 million)
  2. Total Profit for all orders (≈ $286,000)
  3. Average Sales per line item, rounded to 2 decimals
  4. Average Profit per line item, rounded to 2 decimals
  5. Maximum single Sale amount
  6. Minimum Profit value (the biggest loss — should be negative)
  7. Total number of orders using COUNTA on Order ID
  8. Average Quantity per order
  9. Overall profit margin: Total Profit ÷ Total Sales (as a %)
  10. Average discount given across all orders
