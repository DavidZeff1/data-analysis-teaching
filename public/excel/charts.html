📈 Charts & Visualization

A table tells you data exists. A chart tells you what it means. Learn which chart to pick and how to make it communicate clearly.

💡 Why charts matter

  • Reveal patterns — trends, outliers, and clusters become visible
  • Enable comparison — see differences between categories instantly
  • Communicate insights — people read charts faster than tables
  • Drive decisions — well-designed charts influence action

🛠️ Creating a chart

Google Sheets
  1. Select your data (with headers)
  2. Insert → Chart
  3. Customize in the Chart Editor
Excel
  1. Select your data (with headers)
  2. Insert → choose a chart type
  3. Customize with Chart Tools

Pro tip: create a pivot table first, then chart the pivot — the chart updates automatically when you change the pivot.

🎨 Choosing the right chart type

📊

Column / Bar chart

Compare values across categories. Use when: comparing discrete categories (regions, products). Example: Sales by Region.

📈

Line chart

Show trends over time. Use when: data has a time component. Example: monthly sales trend.

🥧 Pie / Donut

Parts of a whole. Use for 2–5 categories adding to 100%.

⚠️ Avoid with many categories — hard to read.

⚬ Scatter plot

Relationship between two variables. Example: Discount vs Profit.

📊📈 Combo chart

Bars + line for two metrics on different scales. Example: Sales (bars) + Profit Margin (line).

📊 Stacked bar

Compare totals while showing their composition. Example: Sales by Region split by Category.

🔗 Chart from a pivot table (best practice)

Create a pivot first, then chart it. Benefits:

  • Chart updates automatically when you modify the pivot
  • Easy to change what's displayed without rebuilding
  • You inherit the pivot's sorting and filtering
  1. Create a pivot table (e.g. Sales by Region)
  2. Select the pivot results (with headers)
  3. Insert → Chart — it's now linked to your pivot

⚙️ Essential customizations

📝 Title

A clear title that tells the story. Good: "Sales by Region (2017)". Bad: "Chart 1".

🏷️ Labels

Add data labels to key values so viewers don't guess.

📐 Axes

Label axes with units. Start the Y-axis at zero for bar charts.

📚 Legend

Position it clear of the data. Remove it for a single series.

🎨 Colors

Be consistent across related charts. Prefer color-blind-friendly palettes.

📏 Gridlines

Reduce or remove for a cleaner look.

🚫 Common chart mistakes

❌ Pie chart with 10+ slices

Impossible to read — use a bar chart.

❌ 3D effects

Distorts perception — keep it flat.

❌ Y-axis not starting at zero

Exaggerates differences (exception: stock prices).

❌ Too many data series

Cluttered — focus on the key insight.

⚡ Sparklines — mini charts

Tiny charts that fit inside a single cell — perfect for dashboards.

Google Sheets: =SPARKLINE(range, {"charttype","line"})   Excel: Insert → Sparklines → choose a type.

=SPARKLINE(B2:M2)                          // line
=SPARKLINE(B2:M2, {"charttype","bar"})    // bar
=SPARKLINE(B2:M2, {"charttype","column"}) // column

🧠 Key intuitions

  • Chart the pivot, not raw data — easier to manage and update
  • One chart, one insight — don't show everything at once
  • Time goes on the X-axis — left to right, past to future
  • Horizontal bars for long labels — product/state names
  • Less is more — remove unnecessary elements
  • Tell a story — the chart should answer a question
  • Test with colleagues — if they don't get it in 5 seconds, simplify

🎯 Practice exercises

  1. Column chart: total Sales by Category
  2. Bar chart: top 10 States by Sales (horizontal)
  3. Line chart: Sales trend by Month (group dates first)
  4. Pie chart: % of Sales by Region
  5. Scatter plot: Discount (X) vs Profit (Y) — what do you see?
  6. Stacked bar: Sales by Region, split by Category
  7. Multi-line: Sales over time for each Region
  8. Combo chart: Sales (bars) + Profit (line) by Sub-Category
  9. Add data labels to your column chart
  10. Chart a pivot table, then modify the pivot and watch it update
  11. Add sparklines for mini sales trends per Region
  12. Build a 3–4 chart dashboard telling one cohesive story
