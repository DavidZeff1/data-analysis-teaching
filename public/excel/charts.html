Charts & Visualization

Why Charts Matter

A table with numbers tells you data exists. A chart tells you what it means. Good visualizations:

  • Reveal patterns - trends, outliers, clusters become visible
  • Enable comparison - see differences between categories instantly
  • Communicate insights - stakeholders understand charts faster than tables
  • Drive decisions - well-designed charts influence action

Creating a Chart

Chart from Pivot Table (Best Practice)

The best approach: Create a pivot table first, then chart it. Benefits:

  • Chart updates automatically when you modify the pivot
  • Easy to change what's displayed without rebuilding
  • Can use pivot's sorting and filtering
  1. Create a pivot table (e.g., Sales by Region)
  2. Select the pivot table results (including headers)
  3. Insert ‚Üí Chart
  4. The chart is now linked to your pivot!

Common Chart Mistakes

‚ùå Pie chart with 10+ slices

Impossible to read. Use bar chart instead.

‚ùå 3D effects

Distorts data perception. Keep it flat.

‚ùå Y-axis not starting at zero

Exaggerates differences. (Exception: stock prices)

‚ùå Too many data series

Chart becomes cluttered. Focus on key insights.

Sparklines - Mini Charts

Sparklines are tiny charts that fit inside a cell - great for dashboards:

Google Sheets: =SPARKLINE(data_range, {"charttype","line"})

Excel: Insert ‚Üí Sparklines ‚Üí Choose type

=SPARKLINE(B2:M2)                           // Line
=SPARKLINE(B2:M2, {"charttype","bar"})     // Bar
=SPARKLINE(B2:M2, {"charttype","column"})  // Column

Key Intuitions

  • Chart the pivot, not raw data - easier to manage and update
  • One chart, one insight - don't try to show everything at once
  • Time goes on X-axis - left to right, past to future
  • Horizontal bars for long labels - product names, state names
  • Less is more - remove unnecessary elements
  • Tell a story - the chart should answer a question
  • Test with colleagues - if they don't understand in 5 seconds, simplify
