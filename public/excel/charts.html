Charts

What they do

Charts visualize data to reveal patterns, trends, and comparisons. Choose the right chart type for your data story.

Creating a Chart

  1. Select your data (including headers)
  2. Google Sheets: Insert → Chart
  3. Excel: Insert → Select chart type
  4. Customize in the Chart Editor panel

Chart Types & When to Use

Column/Bar     - Compare categories
                 Sales by Region, Products by Count

Line           - Show trends over time
                 Monthly Sales, Daily Orders

Pie/Donut      - Show parts of a whole (use sparingly)
                 Market share, % by Category

Scatter        - Show relationship between 2 variables
                 Sales vs Profit, Discount vs Quantity

Area           - Show cumulative trends
                 Running total, Stacked categories over time

Combo          - Mix chart types (bars + line)
                 Sales (bars) with Profit margin (line)

Chart from Pivot Table

Best practice: Create a pivot table first, then chart it.

1. Create pivot: Sales by Region
   Rows: Region
   Values: SUM of Sales

2. Select the pivot table results

3. Insert → Chart → Column chart

4. The chart updates when pivot updates!

Customization Tips

  • Title: Always add a clear, descriptive title
  • Labels: Add data labels for key values
  • Legend: Position where it doesn't overlap data
  • Colors: Use consistent colors across related charts
  • Axis: Start at zero for bar charts (usually)
  • Gridlines: Reduce or remove for cleaner look

Key intuitions

  • Less is more - don't overcrowd charts
  • Pie charts work best with 2-5 categories max
  • Line charts need time-based or sequential data
  • Horizontal bar charts are better for long category names
  • Always label axes and include units
  • Charts from pivot tables are dynamic and update automatically

Practice

  1. Create a column chart showing total Sales by Category
  2. Create a line chart showing Sales over time (by month)
  3. Create a pie chart showing % of Sales by Region
  4. Create a scatter plot with Discount on X-axis and Profit on Y-axis
  5. Build a combo chart: Sales (bars) and Profit (line) by Sub-Category
