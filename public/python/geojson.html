GeoJSON: The Data Lingua Franca

Understanding the JSON format that powers almost all modern web mapping.

1. What is GeoJSON?

GeoJSON is just JSON (JavaScript Object Notation) with a specific rulebook for how to describe shapes on Earth.

  • It is human-readable text.
  • It supports Points, Lines, Polygons.
  • It stores attributes (properties) with each shape.
  • It defaults to WGS84 (EPSG:4326).

The Hierarchy

FeatureCollection: A list of things.
Feature: A single thing (e.g. one park).
Geometry: The shape (coordinates).
Properties: The data (name, area, owner).

2. The Anatomy of GeoJSON

Here is exactly what it looks like. You will likely generate this, not write it by hand. 

{
  "type": "FeatureCollection",
  "features": [
    {
      "type": "Feature",
      "properties": {
        "name": "Central Park",
        "visited": true
      },
      "geometry": {
        "type": "Polygon",
        "coordinates": [
          [
            [-73.958, 40.800],
            [-73.971, 40.764],
            [-73.981, 40.768],
            [-73.958, 40.800]
          ]
        ]
      }
    }
  ]
}

3. Using GeoJSON in Python

Two ways to interact: The "Hard" way (Dictionary) and the "Easy" way (GeoPandas).

Method A: The "Hard" Way (json)

Treat it like a nested dictionary. Good for API responses or simple scripts. 

import json

# Read
with open('data.geojson') as f:
    data = json.load(f)

# Access
first_feature = data['features'][0]
name = first_feature['properties']['name']

Method B: The "Easy" Way (GeoPandas)

Convert immediately to a tabular format. Best for analysis. 

import geopandas as gpd

# Read
gdf = gpd.read_file('data.geojson')

# Write
gdf.to_file('output.geojson', driver='GeoJSON')

4. Common Gotchas

  • XY vs YX: GeoJSON uses [Longitude, Latitude] (X, Y). Google Maps uses (Lat, Lon). This flips often!
  • CRS: GeoJSON spec says coordinates must be WGS84 (lat/lon). If you save projected data (meters) as GeoJSON, web maps will likely put your data in the ocean near Africa (0,0).
  • File Size: GeoJSON is verbose text. Large datasets will result in massive files. Use GeoParquet or GeoPackage for big data.
