Seaborn: Statistical Data Visualization

What is Seaborn?

Seaborn is a Python visualization library built on top of Matplotlib. It provides a high-level interface for creating attractive, informative statistical graphics with minimal code. Think of it as "Matplotlib made easy and beautiful."

üé®
Beautiful Defaults

Attractive color palettes and styles out of the box

üìä
Statistical Focus

Built-in statistical aggregations and confidence intervals

üêº
Pandas Integration

Works seamlessly with DataFrames

üí° Figure-level vs Axes-level:

  • Figure-level (relplot, displot, catplot): Create their own figure, support faceting with col and row
  • Axes-level (scatterplot, histplot, etc.): Draw on current axes, can be combined with matplotlib subplots

1. Relational Plots (relplot)

Visualize relationships between two numeric variables. Use when asking "How does X relate to Y?"

relplot(kind="scatter")
Show individual data points

When to use:

  • ‚Ä¢ Exploring relationship between two continuous variables
  • ‚Ä¢ Looking for correlations, clusters, or outliers
  • ‚Ä¢ Identifying patterns or trends

Key parameters:

  • x, y - Variables for axes
  • hue - Color by category
  • size - Size by numeric variable
  • style - Marker style by category
  • alpha - Transparency (0-1)
# Basic scatter plot
sns.relplot(data=penguins, x="bill_length_mm", y="bill_depth_mm", kind="scatter")

# With color by species and facet by island
sns.relplot(
    data=penguins, 
    x="bill_length_mm", 
    y="bill_depth_mm",
    hue="species",
    col="island",
    kind="scatter"
)

üí° Real-world example: "Is there a relationship between bill length and depth? Does it differ by species or island?"

relplot(kind="line")
Connect points with lines (great for time series)

When to use:

  • ‚Ä¢ Time series data
  • ‚Ä¢ Showing trends over continuous variable
  • ‚Ä¢ Comparing multiple groups over time

Key parameters:

  • x, y - Variables for axes
  • hue - Separate lines by category
  • style - Line style by category
  • markers - Show data points
  • ci - Confidence interval (default: 95)
# Basic line plot
sns.relplot(data=penguins, x="bill_length_mm", y="flipper_length_mm", kind="line")

# Multiple lines by species, faceted by sex
sns.relplot(
    data=penguins, 
    x="bill_length_mm", 
    y="flipper_length_mm",
    hue="species",
    col="sex",
    kind="line"
)

üí° Real-world example: "How does flipper length relate to bill length across species?"

2. Distribution Plots (displot)

Visualize the distribution of a single variable. Use when asking "What does the distribution of X look like?"

displot(kind="hist")
Histogram - bars showing frequency in bins (default)

When to use:

  • ‚Ä¢ Understanding distribution shape
  • ‚Ä¢ Finding central tendency and spread
  • ‚Ä¢ Detecting skewness, outliers, multimodality

Key parameters:

  • x (or y) - Variable to plot
  • bins - Number of bins
  • hue - Separate by category
  • kde - Overlay KDE curve
  • multiple - layer, dodge, stack, fill
# Basic histogram
sns.displot(data=penguins, x="flipper_length_mm", kind="hist")

# Grouped histogram, faceted by sex
sns.displot(
    data=penguins, 
    x="flipper_length_mm",
    hue="species",
    col="sex",
    multiple="stack",
    kind="hist"
)

üí° Real-world example: "What's the distribution of flipper lengths? Does it differ by species or sex?"

displot(kind="kde")
Kernel Density Estimate - smooth distribution curve

When to use:

  • ‚Ä¢ Smooth representation of distribution
  • ‚Ä¢ Comparing multiple distributions
  • ‚Ä¢ 2D density plots (contours)

Key parameters:

  • x (and optionally y) - Variables
  • hue - Separate by category
  • bw_adjust - Bandwidth (smoothness)
  • fill - Fill under curve
# Basic KDE
sns.displot(data=penguins, x="body_mass_g", kind="kde")

# Multiple distributions, faceted by island
sns.displot(
    data=penguins, 
    x="body_mass_g",
    hue="species",
    col="island",
    fill=True,
    kind="kde"
)

üí° Real-world example: "Compare the body mass distribution between species across different islands."

displot(kind="ecdf")
Empirical Cumulative Distribution Function

When to use:

  • ‚Ä¢ Show what percentage of data falls below each value
  • ‚Ä¢ Compare percentiles across groups
  • ‚Ä¢ No binning decisions needed

Key parameters:

  • x - Variable to plot
  • hue - Separate by category
  • stat - proportion (default) or count
  • complementary - Show 1 - CDF
# Basic ECDF
sns.displot(data=penguins, x="flipper_length_mm", kind="ecdf")

# Compare groups, faceted by sex
sns.displot(
    data=penguins, 
    x="flipper_length_mm",
    hue="species",
    col="sex",
    kind="ecdf"
)

üí° Real-world example: "What percentage of penguins have flippers under 200mm? How does this differ by species?"

Read the y-value at x=200 to find the answer.

displot() Faceting
Creating multiple subplots with `col` and `row` 
# displot with histogram (default)
sns.displot(data=penguins, x="flipper_length_mm", hue="species")

# displot with KDE
sns.displot(data=penguins, x="body_mass_g", kind="kde")

# Faceted by column and row
sns.displot(
    data=penguins, 
    x="flipper_length_mm",
    col="island",      # Creates separate columns
    hue="species",
    kind="kde",
    fill=True
)

3. Categorical Plots (catplot)

Visualize relationships between categorical and numeric variables. Use when asking "How does metric Y vary across categories?"

Categorical plot types by purpose:

Show Individual Points

stripplot, swarmplot

See every data point

Show Distribution

boxplot, violinplot

Quartiles, spread, shape

Show Estimate

barplot, pointplot

Central tendency with CI

catplot(kind="strip")
Scatter points by category (default)

When to use:

  • ‚Ä¢ See every individual data point
  • ‚Ä¢ Small to medium datasets
  • ‚Ä¢ Spot outliers

Key parameters:

  • x, y - Categorical and numeric
  • hue - Sub-categories
  • jitter - Spread (True/False or float)
  • dodge - Separate hue groups
# Basic strip plot
sns.catplot(data=penguins, x="species", y="body_mass_g", kind="strip")

# With hue and faceting
sns.catplot(
    data=penguins, 
    x="species", 
    y="body_mass_g",
    hue="sex",
    col="island",
    kind="strip"
)
catplot(kind="swarm")
Non-overlapping points (beeswarm)

When to use:

  • ‚Ä¢ See every point without overlap
  • ‚Ä¢ Small datasets only (slow for large n)
  • ‚Ä¢ Shows distribution shape through point density

Key parameters:

  • x, y - Categorical and numeric
  • hue - Sub-categories
  • dodge - Separate hue groups
  • size - Point size
# Basic swarm plot
sns.catplot(data=penguins, x="species", y="bill_depth_mm", kind="swarm")

# With hue and faceting
sns.catplot(
    data=penguins, 
    x="species", 
    y="bill_depth_mm",
    hue="sex",
    col="island",
    kind="swarm"
)

‚ö†Ô∏è Warning: swarmplot can be very slow with large datasets (>500 points). Use stripplot instead.

catplot(kind="box")
Box and whisker - shows quartiles

What it shows:

  • ‚Ä¢ Box: IQR (Q1 to Q3, middle 50%)
  • ‚Ä¢ Line in box: Median (Q2)
  • ‚Ä¢ Whiskers: Q1 - 1.5√óIQR to Q3 + 1.5√óIQR
  • ‚Ä¢ Diamonds: Outliers beyond whiskers

When to use:

  • ‚Ä¢ Compare distributions across categories
  • ‚Ä¢ Quick summary of center and spread
  • ‚Ä¢ Identify outliers
# Basic boxplot
sns.catplot(data=penguins, x="species", y="bill_length_mm", kind="box")

# With hue and faceting
sns.catplot(
    data=penguins, 
    x="species", 
    y="bill_length_mm",
    hue="sex",
    col="island",
    kind="box"
)

üí° Real-world example: "Compare bill lengths by species. Which species has the longest bills? Most variability?"

catplot(kind="violin")
Distribution shape + boxplot combined

When to use:

  • ‚Ä¢ See full distribution shape (not just quartiles)
  • ‚Ä¢ Detect bimodality or skewness
  • ‚Ä¢ More informative than boxplot

Key parameters:

  • x, y - Categorical and numeric
  • hue - Sub-categories
  • split - Split violin by hue (great for comparison)
  • inner - "box", "quartile", "point", "stick", None
# Basic violin plot
sns.catplot(data=penguins, x="species", y="flipper_length_mm", kind="violin")

# Split violin for comparison, faceted by island
sns.catplot(
    data=penguins, 
    x="species", 
    y="flipper_length_mm",
    hue="sex",
    split=True,
    col="island",
    kind="violin"
)

üí° Real-world example: "Compare flipper length distributions between male and female penguins. Is the shape different?"

catplot(kind="bar")
Bar height = mean (or other stat) with confidence interval

When to use:

  • ‚Ä¢ Compare averages across categories
  • ‚Ä¢ Show statistical uncertainty (error bars)
  • ‚Ä¢ Familiar format for business audiences

Key parameters:

  • x, y - Categorical and numeric
  • hue - Sub-categories
  • estimator - Function to aggregate (default: mean)
  • ci - Confidence interval (default: 95)
  • order - Category order
# Basic bar plot (shows mean + CI)
sns.catplot(data=penguins, x="species", y="body_mass_g", kind="bar")

# With hue and faceting
sns.catplot(
    data=penguins, 
    x="species", 
    y="body_mass_g",
    hue="sex",
    col="island",
    kind="bar"
)

‚ö†Ô∏è Note: seaborn barplot shows AGGREGATED values (mean by default), not raw counts. For counts, use countplot().

catplot(kind="point")
Points + lines showing mean with CI - great for interactions

When to use:

  • ‚Ä¢ Compare trends across categories
  • ‚Ä¢ Show interactions between variables
  • ‚Ä¢ Connect related categories visually

Key parameters:

  • x, y - Categorical and numeric
  • hue - Separate lines
  • estimator - Function to aggregate
  • ci - Confidence interval
  • markers, linestyles - Customize appearance
# Basic point plot
sns.catplot(data=penguins, x="species", y="body_mass_g", kind="point")

# With hue - shows interaction, faceted by island
sns.catplot(
    data=penguins, 
    x="species", 
    y="body_mass_g",
    hue="sex",
    col="island",
    markers=["^", "o"],
    linestyles=["-", "--"],
    kind="point"
)

üí° Real-world example: "Does the species effect on body mass differ by sex?" (If lines cross or have different slopes = interaction)

catplot(kind="count")
Bar chart of counts (like value_counts visualized) 
# Count occurrences of each category
sns.catplot(data=penguins, x="species", kind="count")

# With hue and faceting
sns.catplot(data=penguins, x="species", hue="sex", col="island", kind="count")

# Horizontal
sns.catplot(data=penguins, y="island", kind="count")
catplot() Faceting
Creating multiple subplots with `col` and `row` 
# catplot with different kinds
sns.catplot(data=penguins, x="species", y="body_mass_g", kind="box")
sns.catplot(data=penguins, x="species", y="body_mass_g", kind="violin")
sns.catplot(data=penguins, x="species", y="body_mass_g", kind="bar")

# Faceted by island
sns.catplot(
    data=penguins, 
    x="species", 
    y="body_mass_g",
    hue="sex",
    col="island",      # Separate plots for each island
    kind="box",
    height=4
)

4. Other Useful Plots

heatmap()
Color-coded matrix - great for correlations 
# Correlation heatmap
corr = penguins[["bill_length_mm", "bill_depth_mm", "flipper_length_mm", "body_mass_g"]].corr()
sns.heatmap(corr, annot=True, cmap="coolwarm", center=0)

# Pivot table heatmap
pivot = penguins.pivot_table(values="body_mass_g", index="species", columns="island")
sns.heatmap(pivot, annot=True, fmt=".0f", cmap="YlGnBu")

# Customize
sns.heatmap(
    corr, 
    annot=True,        # Show values
    fmt=".2f",         # Number format
    cmap="RdBu_r",     # Color palette
    center=0,          # Center color at 0
    vmin=-1, vmax=1,   # Value range
    square=True,       # Square cells
    linewidths=0.5     # Grid lines
)
pairplot()
Matrix of scatter plots for all variable pairs 
# Basic pairplot
sns.pairplot(penguins)

# With hue (color by species)
sns.pairplot(penguins, hue="species")

# Select columns
sns.pairplot(penguins, vars=["bill_length_mm", "bill_depth_mm", "flipper_length_mm"], hue="species")

üí° Use case: Quick EDA to see relationships between all numeric variables at once. Great for finding correlations and clusters.

jointplot()
Bivariate plot with marginal distributions 
# Basic jointplot (scatter + histograms)
sns.jointplot(data=penguins, x="bill_length_mm", y="bill_depth_mm")

# Different kinds
sns.jointplot(data=penguins, x="bill_length_mm", y="bill_depth_mm", kind="kde")

# With hue
sns.jointplot(data=penguins, x="bill_length_mm", y="bill_depth_mm", hue="species")
regplot() / lmplot()
Scatter with regression line 
# Simple regression plot
sns.regplot(data=penguins, x="bill_length_mm", y="bill_depth_mm")

# lmplot (figure-level, supports faceting)
sns.lmplot(data=penguins, x="bill_length_mm", y="bill_depth_mm", hue="species")

# Faceted
sns.lmplot(
    data=penguins, 
    x="bill_length_mm", 
    y="bill_depth_mm",
    col="island",
    hue="species"
)

Figure-level vs Axes-level

Figure-level (creates own figure)

  • relplot() - relational
  • displot() - distributions
  • catplot() - categorical
  • lmplot() - regression
  • pairplot() - all pairs
  • jointplot() - bivariate + marginals

‚úì Support col and row for faceting

‚úì Use height and aspect for size

Axes-level (draws on current axes)

  • scatterplot(), lineplot()
  • histplot(), kdeplot(), ecdfplot()
  • stripplot(), swarmplot(), boxplot()
  • violinplot(), barplot(), pointplot()
  • heatmap(), regplot()

‚úì Can combine with matplotlib subplots

‚úì Use plt.figure(figsize=()) for size

Common Parameters

Data & Variables

  • data - DataFrame
  • x, y - Column names
  • hue - Color by category
  • size - Size by variable (scatter)
  • style - Marker/line style by category

Faceting (figure-level only)

  • col - Create columns of plots
  • row - Create rows of plots
  • col_wrap - Wrap columns after N

Sizing

  • height - Figure height (figure-level)
  • aspect - Width = height √ó aspect
  • plt.figure(figsize=(w,h)) - axes-level

Appearance

  • palette - Color palette
  • alpha - Transparency (0-1)
  • order - Category order
  • hue_order - Hue category order

Example EDA Workflow

import seaborn as sns
import matplotlib.pyplot as plt

# Load data
penguins = sns.load_dataset("penguins")

# 1. Quick overview with pairplot
sns.pairplot(penguins, hue="species")
plt.show()

# 2. Distribution of main metric
fig, axes = plt.subplots(1, 2, figsize=(12, 4))
sns.histplot(data=penguins, x="body_mass_g", kde=True, ax=axes[0])
sns.boxplot(data=penguins, x="body_mass_g", ax=axes[1])
plt.tight_layout()
plt.show()

# 3. Compare across categories
sns.catplot(
    data=penguins, 
    x="species", 
    y="body_mass_g",
    hue="sex",
    col="island",
    kind="violin",
    split=True
)
plt.show()

# 4. Correlation heatmap
plt.figure(figsize=(8, 6))
cols = ["bill_length_mm", "bill_depth_mm", "flipper_length_mm", "body_mass_g"]
sns.heatmap(penguins[cols].corr(), annot=True, cmap="coolwarm", center=0)
plt.title("Correlation Matrix")
plt.show()

# 5. Key relationship with regression
sns.lmplot(data=penguins, x="bill_length_mm", y="bill_depth_mm", hue="species", col="island")
plt.show()
