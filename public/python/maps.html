Geospatial Visualization: The Comprehensive Guide

Mastering the art of layering data: From raw CSVs to complex multi-layered interactive maps.

0. The Geospatial Pipeline: Getting Data Map-Ready

Geospatial data almost never starts in a "map-ready" format. Usually, you have a CSV, an Excel sheet, or a list of addresses. Your job is to convert that raw data into a GeoDataFrame.

Input Format What it looks like Required Action
CSV / Excel Columns: lat, lon Create Geometry from XY points
Addresses String: "123 Main St..." Geocoding (API request)
Database / WKT String: "POINT(12 34)" Parse WKT string to Geometry
Shapefile / GeoJSON Standard file formats Read directly with GeoPandas

Case 1: CSV / Excel with Lat/Lon

The most common scenario for business data.

import pandas as pd
import geopandas as gpd

# 1. Load raw data
df = pd.read_csv('stores.csv')  # Or pd.read_excel()

# 2. Convert to GeoDataFrame
gdf = gpd.GeoDataFrame(
    df, 
    geometry=gpd.points_from_xy(df.longitude, df.latitude),
    crs="EPSG:4326"
)

Case 2: Raw Addresses (Geocoding)

Turning "1600 Amphitheatre Pkwy" into coordinates.

from geopandas.tools import geocode

# Provide a list of addresses
addresses = ["Empire State Building", "Eiffel Tower"]

# Geocode using Photon (free) or Nominatim
gdf = geocode(addresses, provider='photon')

# Result is already a GeoDataFrame with 'geometry' and 'address'
Warning: Geocoding is slow and usually has API limits. Batch your requests!

Case 3: WKT Strings (from Databases)

Often used in PostGIS or BigQuery exports.

from shapely import wkt

# df['geo_boundary'] = "POLYGON ((...))"
df['geometry'] = df['geo_boundary'].apply(wkt.loads)
gdf = gpd.GeoDataFrame(df, crs="EPSG:4326")

Case 4: Direct Loading (GeoJSON / Shapefile)

When the data is already geospatial. No conversion needed!

# Read practically any spatial file format
gdf = gpd.read_file('neighborhoods.geojson') 
# gdf = gpd.read_file('states.shp')

1. Mastering Layers (Static Maps)

A map is just a stack of transparencies. You build it from the bottom up.

The Pattern: Points on Polygons

Common task: Show customer locations (Points) on top of neighborhood boundaries (Polygons). 

import matplotlib.pyplot as plt

# 1. Setup the canvas (Figure & Axis)
fig, ax = plt.subplots(figsize=(12, 12))

# 2. Bottom Layer: Neighborhood Polygons
neighborhoods.plot(
    ax=ax,
    color='whitesmoke',      # Light background
    edgecolor='lightgrey',   # Subtle borders
    zorder=1                 # Draw first (bottom)
)

# 3. Top Layer: Store Points
gdf_stores.plot(
    ax=ax,
    color='red',           # Standout color
    markersize=20,         # Adjust size
    alpha=0.7,             # Slight transparency for overlapping points
    zorder=2               # Draw second (top)
)

# 4. Final Polish
ax.set_axis_off()          # Hides the box/ticks
plt.title("Store Locations in Chicago")

Categorical Plots

Color points by a category column (e.g. 'Type': Grocery, Clothes, Electronics). 

gdf_stores.plot(
    column='Type',   # Column to color by
    ax=ax,
    legend=True,     # Add legend automatically
    cmap='Set1'      # Qualitative colormap
)

2. Choropleth Maps (Heatmaps)

The classic "Color by Number". Best for visualizing density or average values across regions. 

neighborhoods.plot(
    column='population_density',
    legend=True,
    cmap='YlOrRd',            # Yellow to Red
    scheme='quantiles',       # Binning (requires mapclassify)
    k=5,                      # 5 Buckets
    edgecolor='black',        # separation lines
    linewidth=0.1
)

3. Advanced Interactive Maps (Folium)

Static maps are for slides. Interactive maps are for exploration.

The Base Map

import folium

m = folium.Map(
    location=[41.87, -87.62],
    zoom_start=11,
    tiles='CartoDB positron'
)

The Layer Control

# Add this LAST after adding all layers
folium.LayerControl().add_to(m)

Building a Multi-Layer Interactive Map

Combine polygons and points with toggles. 

# 1. Create Base Map
m = folium.Map(location=[41.87, -87.62], zoom_start=11)

# 2. Add Choropleth Layer (Neighborhoods)
folium.Choropleth(
    geo_data=neighborhoods,
    name='Population Density',
    data=neighborhoods,
    columns=['name', 'density'],
    key_on='feature.properties.name',
    fill_color='YlOrRd'
).add_to(m)

# 3. Add Point Layer (Stores) using GeoJson
# We use GeoJson to add tooltips and custom styles easily
folium.GeoJson(
    gdf_stores,
    name='Store Locations',
    tooltip=folium.GeoJsonTooltip(fields=['StoreName', 'Type']),
    marker=folium.CircleMarker(radius=4, fill_color='blue')
).add_to(m)

# 4. Add Toggle Control
folium.LayerControl().add_to(m)

4. Contextily (Basemaps)

For static maps only. Adds the background map tiles. 

import contextily as cx

# ⚠️ CRITICAL: Must be in Web Mercator (EPSG:3857)
ax = gdf.to_crs(epsg=3857).plot(figsize=(10, 10), alpha=0.5)

# Add the basemap
cx.add_basemap(ax, source=cx.providers.CartoDB.Positron)
