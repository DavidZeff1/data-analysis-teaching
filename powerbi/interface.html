Building a Visual

// Method 1: Click and drag
1. Click a chart type in Visualizations pane
2. Drag fields from Fields pane to the wells

// Method 2: Quick create
1. Check boxes next to fields in Fields pane
2. Power BI auto-creates a visual

// Example: Sales by Category bar chart
1. Click "Clustered bar chart" icon
2. Drag Category to Y-axis
3. Drag Sales to X-axis

Field Wells

For a bar chart:
Y-axis Categories (what you're comparing)
X-axis Values (numbers)
Legend Color/split bars
Tooltips Extra info on hover
Wells change based on visual type!

Key intuitions

  • Click empty canvas, then choose visual type
  • Click existing visual to select and modify it
  • Drag edges to resize visuals
  • Format pane (paint roller) for styling
  • Ctrl+Click to select multiple visuals
  • Right-click for more options

Practice

  1. Create a bar chart showing Sales by Region
  2. Add Category to the Legend to split bars by category
  3. Resize the visual to be larger
  4. Use the Format pane to change bar colors
  5. Switch between Report, Table, and Model views
