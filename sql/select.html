SELECT Statement

What it does

SELECT retrieves data from a table. Think of it as asking a question to your database.

Syntax

SELECT column1, column2 FROM table_name;

Examples

-- Get all columns
SELECT * FROM Artist;

-- Get specific columns
SELECT Name FROM Artist;

-- Rename columns with AS
SELECT Name AS ArtistName FROM Artist;

Key intuitions

  • * means "everything" - quick but avoid in production
  • Column order in SELECT = order in results
  • AS renames output columns without changing the table
