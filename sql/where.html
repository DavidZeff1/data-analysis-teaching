WHERE Clause

What it does

WHERE filters rows based on a condition. Only rows that match are returned.

Syntax

SELECT columns FROM table WHERE condition;

Operators

=    Equal
<>   Not equal
>    Greater than
<    Less than
>=   Greater or equal
<=   Less or equal
AND  Both conditions true
OR   Either condition true
IN   Matches any in list
LIKE Pattern matching (% = any chars, _ = one char)
BETWEEN  Within a range

Examples

-- Exact match
SELECT * FROM Customer WHERE Country = 'USA';

-- Numeric comparison
SELECT * FROM Track WHERE UnitPrice > 0.99;

-- Multiple conditions
SELECT * FROM Track WHERE UnitPrice > 0.99 AND Milliseconds > 300000;

-- Pattern matching
SELECT * FROM Artist WHERE Name LIKE 'The%';

-- Match from a list
SELECT * FROM Customer WHERE Country IN ('USA', 'Canada', 'Brazil');

-- Range
SELECT * FROM Invoice WHERE Total BETWEEN 5 AND 10;

Key intuitions

  • WHERE runs before SELECT - it filters first, then picks columns
  • Strings need quotes, numbers don't
  • LIKE is case-insensitive in SQLite
  • Use IS NULL or IS NOT NULL for null checks (not = NULL)

Practice

  1. Get all customers from Germany
  2. Get all tracks longer than 5 minutes (300000 milliseconds)
  3. Get all employees whose Title contains 'Manager'
  4. Get all invoices with Total greater than 10 from USA
  5. Get all artists whose name starts with 'A'
