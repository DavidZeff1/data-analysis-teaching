Lookup Functions

What they do

Lookup functions search for a value and return related data from another column or row. Essential for connecting data.

Key Functions

VLOOKUP(search, range, col_num, exact)   - Vertical lookup
HLOOKUP(search, range, row_num, exact)   - Horizontal lookup
INDEX(range, row, col)                   - Return value at position
MATCH(search, range, type)               - Find position of value
XLOOKUP(search, lookup_range, return_range) - Modern lookup

Examples

// VLOOKUP - find customer name by ID
=VLOOKUP("CG-12520", F:G, 2, FALSE)

// VLOOKUP breakdown:
//   "CG-12520" = value to find
//   F:G = range to search (ID in F, Name in G)
//   2 = return 2nd column (Name)
//   FALSE = exact match

// INDEX + MATCH - more flexible than VLOOKUP
=INDEX(G:G, MATCH("CG-12520", F:F, 0))

// INDEX + MATCH breakdown:
//   MATCH finds the row number
//   INDEX returns value from that row

// XLOOKUP - simplest (Google Sheets & Excel 365)
=XLOOKUP("CG-12520", F:F, G:G)

// Lookup with default value if not found
=XLOOKUP("Unknown", F:F, G:G, "Not Found")

// Lookup from right to left (VLOOKUP can't do this)
=INDEX(F:F, MATCH("Aaron Bergman", G:G, 0))

Key intuitions

  • VLOOKUP searches the first column of your range
  • Always use FALSE (or 0) for exact match
  • INDEX+MATCH is more flexible - can look left, right, any direction
  • XLOOKUP is the modern replacement - simpler syntax
  • If lookup returns #N/A, the value wasn't found
  • Wrap in IFERROR to handle not-found cases

VLOOKUP vs INDEX+MATCH

VLOOKUP limitations:
- Can only search leftmost column
- Column number breaks if columns inserted
- Slower on large datasets

INDEX+MATCH advantages:
- Search any column, return any column
- More flexible and robust
- Faster on large datasets

Practice

  1. Use VLOOKUP to find the Category for Product ID "OFF-BI-10000756"
  2. Use INDEX+MATCH to find the Customer Name for Order ID "CA-2016-152156"
  3. Use XLOOKUP to find the Region for a given City
  4. Create a summary sheet that looks up totals by Region
  5. Handle errors: wrap a lookup in IFERROR to show "Not Found" for missing values
