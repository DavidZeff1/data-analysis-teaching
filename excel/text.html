Text Functions

What they do

Text functions manipulate strings - extract parts, combine text, change case, find patterns, and clean data.

Key Functions

LEFT(text, num)         - First n characters
RIGHT(text, num)        - Last n characters
MID(text, start, num)   - Characters from middle
LEN(text)               - Length of text
FIND(find, text)        - Position of substring
CONCAT(text1, text2)    - Join text
TEXTJOIN(delim, ignore_empty, range)
TRIM(text)              - Remove extra spaces
UPPER/LOWER/PROPER     - Change case
SUBSTITUTE(text, old, new)
TEXT(value, format)     - Format numbers as text

Examples

Using Superstore: Order ID (B), Customer Name (G), Product Name (R)

// Extract state code from Order ID "CA-2016-152156"
=LEFT(B2, 2)                    // Returns "CA"

// Extract year from Order ID
=MID(B2, 4, 4)                  // Returns "2016"

// Get first name from "Claire Gute"
=LEFT(G2, FIND(" ", G2)-1)      // Returns "Claire"

// Get last name
=RIGHT(G2, LEN(G2)-FIND(" ", G2))

// Combine text
=CONCAT(G2, " - ", L2)         // "Claire Gute - East"
=G2&" - "&L2                     // Same with & operator

// Clean messy data
=TRIM(G2)                       // Remove extra spaces
=UPPER(G2)                      // "CLAIRE GUTE"
=PROPER(G2)                     // "Claire Gute"

// Replace text
=SUBSTITUTE(K2, "United States", "USA")

// Format number as text
=TEXT(N2, "$#,##0.00")         // "$1,234.56"
=TEXT(C2, "MMMM YYYY")         // "January 2016"

Key intuitions

  • FIND returns position (1-based), use it with LEFT/MID/RIGHT
  • Use & operator for quick concatenation
  • TRIM removes leading, trailing, and extra internal spaces
  • TEXT is powerful for formatting dates and numbers
  • SUBSTITUTE replaces all occurrences, REPLACE replaces by position
