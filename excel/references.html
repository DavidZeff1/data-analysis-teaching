Cell References

What they do

References tell formulas which cells to use. Understanding relative vs absolute references is crucial for copying formulas.

Types of References

A1      - Relative: changes when copied
$A$1    - Absolute: stays fixed when copied
$A1     - Mixed: column fixed, row changes
A$1     - Mixed: row fixed, column changes

Examples

// Relative reference - changes when copied down
=N2*O2          // Becomes N3*O3, N4*O4, etc.

// Absolute reference - stays fixed
=N2/$N$9995    // Always divides by the total in N9995

// Mixed reference - useful for tables
=$A2*B$1        // Column A fixed, Row 1 fixed

// Calculating % of total sales
=N2/SUM($N$2:$N$9995)

// Reference another sheet
=Sheet2!A1
='Sheet Name With Spaces'!A1

Key intuitions

  • $ locks the part that follows it
  • Press F4 to cycle through reference types
  • Use absolute references when referring to a fixed cell (like a total)
  • Use relative references when you want the formula to adjust as you copy
  • Named ranges (Insert → Named ranges) can replace complex references

Visual Example

If you copy =A1+B1 from row 1 to row 2:

Row 1: =A1+B1
Row 2: =A2+B2  ← adjusted automatically

If you copy =$A$1+B1 from row 1 to row 2:
Row 1: =$A$1+B1
Row 2: =$A$1+B2  ← A1 stays fixed, B adjusts

Practice

  1. In a new column, calculate each row's Sales as a % of total Sales (use absolute reference for total)
  2. Create a formula that calculates Profit Margin (Profit/Sales) and copy it down
  3. Reference a cell from the first sheet in a second sheet
  4. Create a named range for the Sales column and use it in a SUM formula
