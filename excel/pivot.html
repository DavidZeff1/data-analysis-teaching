Pivot Tables

What they do

Pivot tables summarize large datasets by grouping, filtering, and calculating totals. They're the most powerful tool for data analysis in Excel.

Creating a Pivot Table

  1. Select your data (including headers)
  2. Google Sheets: Insert → Pivot table
  3. Excel: Insert → PivotTable
  4. Choose where to place it (new sheet recommended)

Pivot Table Areas

Rows      - Categories to group by (vertical)
Columns   - Categories to spread across (horizontal)  
Values    - Numbers to calculate (SUM, COUNT, AVG)
Filters   - Limit data shown

Example Pivot Tables

// Total Sales by Region
Rows: Region
Values: SUM of Sales

// Sales by Region and Category
Rows: Region
Columns: Category
Values: SUM of Sales

// Average Profit by Segment and Ship Mode
Rows: Segment
Columns: Ship Mode
Values: AVERAGE of Profit

// Order Count by Year and Quarter
Rows: Order Date (grouped by Year, Quarter)
Values: COUNT of Order ID

// Top Products by Sales (filtered)
Rows: Product Name
Values: SUM of Sales
Filters: Category = "Technology"
Sort: Descending by Sales

Key Features

  • Group dates: Right-click date field → Group by Month/Quarter/Year
  • Sort: Click dropdown on row/column headers
  • Filter: Add fields to Filter area or use dropdown arrows
  • Show as %: Value field settings → Show values as → % of Grand Total
  • Multiple values: Drag same field to Values twice with different calculations

Key intuitions

  • Pivot tables don't change your original data
  • Refresh pivot table when source data changes
  • Text fields go in Rows/Columns, number fields go in Values
  • Start simple (one row, one value) then add complexity
  • Double-click any cell to see the underlying rows
