Practice Setup

1. Open Google Sheets

Go to sheets.new (creates a new blank spreadsheet)

Google Sheets is free and works like Excel. All formulas are the same.

2. Import the Sample Superstore dataset

In cell A1, paste this formula and press Enter:

=IMPORTDATA("https://raw.githubusercontent.com/yannie28/Global-Superstore/master/Global_Superstore_100.csv")

Wait a few seconds for the data to load (~10,000 rows).

3. Dataset overview

The Sample Superstore is a classic dataset for learning data analysis. It contains sales data for a fictional retail company.

Columns:

  • Row ID - Unique row identifier
  • Order ID - Unique order identifier
  • Order Date / Ship Date - When ordered and shipped
  • Ship Mode - Standard, Second Class, First Class, Same Day
  • Customer ID / Customer Name - Customer info
  • Segment - Consumer, Corporate, Home Office
  • Country / City / State / Region - Location
  • Product ID / Category / Sub-Category / Product Name - Product info
  • Sales - Revenue from the sale
  • Quantity - Number of items
  • Discount - Discount applied (0 to 0.8)
  • Profit - Profit or loss from the sale

4. Test it

After import, you should see data starting with headers in row 1. Try these to verify:

  • Scroll down - you should have ~9,994 rows of data
  • Column A should be "Row ID"
  • Column N should be "Sales"
