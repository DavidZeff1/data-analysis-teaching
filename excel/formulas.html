Basic Formulas

What they do

Formulas perform calculations on your data. They always start with =

Common Functions

SUM(range)      - Add all values
AVERAGE(range)  - Calculate mean
COUNT(range)    - Count numbers
COUNTA(range)   - Count non-empty cells
MIN(range)      - Smallest value
MAX(range)      - Largest value
ROUND(num, decimals) - Round a number

Examples

Using the Superstore dataset (Sales is column N, Quantity is column O):

// Total sales
=SUM(N:N)

// Average sale amount
=AVERAGE(N:N)

// Number of orders
=COUNTA(B2:B9995)

// Highest single sale
=MAX(N:N)

// Lowest profit (might be negative)
=MIN(Q:Q)

// Average rounded to 2 decimals
=ROUND(AVERAGE(N:N), 2)

// Basic arithmetic
=N2*O2           // Multiply cells
=N2+Q2           // Add cells
=(N2-Q2)/N2      // Calculate margin

Key intuitions

  • Formulas always start with =
  • N:N means entire column N, N2:N100 means specific range
  • COUNT only counts numbers, COUNTA counts any non-empty cell
  • You can nest functions: =ROUND(AVERAGE(N:N), 2)
  • Use column letters and row numbers to reference cells
