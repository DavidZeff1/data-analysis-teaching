Interface Basics

Key Concepts

Dimensions  - Categorical data (text, dates, IDs)
             Shown in blue, used for grouping

Measures    - Numeric data (sales, quantity, profit)
             Shown in green, used for calculations

Pills       - Draggable fields that build your view

Shelves     - Drop zones for pills (Rows, Columns, etc.)

Building a View

// Basic bar chart: Sales by Category
1. Drag Category to Rows
2. Drag Sales to Columns
   ‚Üí Horizontal bar chart!

// Add color by Region
3. Drag Region to Color on Marks card
   ‚Üí Stacked colored bars!

// Swap axes
4. Click swap button (or Ctrl+W)
   ‚Üí Vertical bar chart!

Key intuitions

  • Rows = Y-axis, Columns = X-axis
  • Drag and drop is the primary interaction
  • Right-click anything for more options
  • Ctrl+Z to undo (use it liberally!)
  • Double-click a measure to auto-create a view
  • Show Me (top right) suggests chart types

Practice

  1. Create a bar chart showing Sales by Region
  2. Add color to show Segment within each region
  3. Swap the axes to make it vertical
  4. Add labels to show the sales values
  5. Explore the Show Me panel - try different chart types
